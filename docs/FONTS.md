# Bundled font provenance

`app/src/main/res/font/talking_clock_seven_segment.ttf` is derived from
`DSEG7Classic-Regular.ttf` in the official DSEG v0.46 release:
https://github.com/keshikan/DSEG/releases/tag/v0.46

- Upstream release ZIP SHA-256: `A6C2F43520971CA8067262E78D49025E605F749BF716EC5394BAD9A0EE1C238C`
- Upstream TTF SHA-256: `E3270928CED68082F32D3B62877A6741426116335AC0832919326341A8B5CFAF`
- Bundled TTF SHA-256: `75B72857B22751F554A13CED62CF35643E738DAECA0723CA886889B83E81239F`

The bundled file was produced with FontTools 4.63.0 by retaining Unicode
`U+0030–U+003A` (digits and colon) and all name/license records. Because
“DSEG” is a Reserved Font Name under the upstream OFL license, name IDs 1,
3, 4, and 6 were changed to “Talking Clock Seven Segment” equivalents.
The glyph outlines are otherwise unchanged. License metadata is in
`.reuse/dep5`; the full license is `LICENSES/OFL-1.1.txt`.
